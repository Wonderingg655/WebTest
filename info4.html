บทที่ 4 การจัดรูปแบบเว็บเพจด้วยภาษา HTML และภาษา CSS
  การจัดรูปแบบเว็บเพจด้วยภาษา HTML และ CSS คือการใช้ HTML ในการสร้างโครงสร้างของเนื้อหา เช่น หัวข้อ ย่อหน้า ลิงก์ และรูปภาพ ขณะที่ CSS ใช้ในการตกแต่งรูปลักษณ์ของเว็บเพจ เช่น การกำหนดสี ขนาดฟอนต์ การจัดตำแหน่ง และการจัดรูปแบบอื่นๆ เพื่อทำให้เว็บเพจดูสวยงามและใช้งานง่าย โดยการเชื่อมโยง HTML กับ CSS ผ่านการใช้แอตทริบิวต์ใน HTML หรือการเชื่อมโยงไฟล์ CSS ภายนอก เพื่อให้สามารถควบคุมและปรับแต่งการแสดงผล
จุดประสงค์การเรียนรู้
  1. สามารถการจัดรูปแบบข้อความบนหน้าเว็บเพจได้
  2. สามารถการจัดรูปแบบรูปภาพบนหน้าเว็บเพจได้
