1.1 อินเทอร์เน็ตและเวิลด์ไวด์เว็บ
อินเทอร์เน็ต คืออะไร
   อินเทอร์เน็ตคือเครือข่ายขนาดใหญ่ที่เชื่อมโยงคอมพิวเตอร์และอุปกรณ์ต่างๆ ทั่วโลกเข้าด้วยกันผ่านระบบการสื่อสารข้อมูล ซึ่งช่วยให้ผู้คนแลกเปลี่ยนข้อมูล ส่งข้อความ ค้นหาความรู้ และใช้บริการต่างๆ เช่น อีเมล โซเชียลมีเดีย และการซื้อขายออนไลน์ได้อย่างสะดวกรวดเร็ว และมีบทบาทสำคัญในชีวิตประจำวัน รวมถึงการพัฒนาด้านเศรษฐกิจ การศึกษา และเทคโนโลยีในสังคมสมัยใหม่

เวิลด์ไวด์เว็บ คืออะไร
  เวิลด์ไวด์เว็บ WWW หรือ World Wide Web คือระบบข้อมูลที่เชื่อมโยงกันบนอินเทอร์เน็ต ซึ่งช่วยให้ผู้ใช้งานสามารถเข้าถึงและแลกเปลี่ยนข้อมูลผ่านหน้าเว็บเพจที่ออกแบบด้วยภาษา HTML โดยใช้โปรโตคอล HTTP ในการส่งข้อมูล เว็บเพจเหล่านี้สามารถเข้าถึงได้ผ่านโปรแกรมเว็บเบราว์เซอร์และมีการเชื่อมโยงกันด้วยลิงก์ ทำให้ผู้ใช้งานสามารถค้นหาและเข้าถึงข้อมูลต่างๆ ได้อย่างง่ายดาย WWW เป็นส่วนสำคัญของอินเทอร์เน็ตที่เปลี่ยนแปลงการสื่อสารและการแบ่งปันข้อมูลในยุคดิจิทัลอย่างมาก
