1.3 รู้จักภาษา HTML
ภาษา HTML คืออะไร
   HTML (HyperText Markup Language) คือภาษามาร์กอัปที่ใช้สร้างเว็บไซต์ โดย HTML จะใช้ชุดของ แท็ก (tags) และ แอตทริบิวต์ (attributes) เพื่อแยกแยะและกำหนดลำดับหรือโครงสร้างของเนื้อหาแต่ละส่วนบนหน้าเว็บ เช่น ข้อความ, ลิงก์, รูปภาพ เพื่อให้เว็บเบราว์เซอร์แสดงผลได้ HTML ทำงานร่วมกับ CSS เพื่อการออกแบบ และ JavaScript เพื่อเพิ่มฟังก์ชันการทำงาน ภาษา HTML เป็นภาษาพื้นฐานสำคัญของการพัฒนาเว็บเพจในโลกออนไลน์ โดยปัจจุบันใช้รุ่น HTML 5 เป็นมาตรฐาน
แท็กในภาษา HTML
  แท็ก (Tag) คือคำสั่งที่ใช้ในการกำหนดและจัดรูปแบบองค์ประกอบต่างๆ บนเว็บเพจ โดยแต่ละแท็กจะมีการเปิดและปิด เช่น <h1> </h1>, <p> </p>, <div> </div> แท็กช่วยให้เว็บเบราว์เซอร์สามารถรู้ว่าเนื้อหานั้นๆ ควรแสดงผลในรูปแบบไหน หรือมีฟังก์ชันอะไร ซึ่งแท็กบางตัวอาจมีแอตทริบิวต์เพิ่มเติมเพื่อกำหนดลักษณะเฉพาะ เช่น ขนาด สี หรือการเชื่อมโยงกับข้อมูลอื่น ๆ
แอตทริบิวต์ในภาษา HTML
แอตทริบิวต์ (Attribute) คือคุณสมบัติที่ใช้ระบุข้อมูลเพิ่มเติมให้กับแท็ก เพื่อกำหนดลักษณะหรือพฤติกรรมขององค์ประกอบในเว็บเพจ โดยแอตทริบิวต์จะถูกเพิ่มไว้ในแท็กเปิดและประกอบด้วยชื่อและค่าที่กำหนด เช่น id, class, src, หรือ href เช่น <img src="image.jpg" alt="รูปภาพ">` ซึ่ง src ระบุแหล่งที่มาของรูปภาพและ alt ระบุข้อความแสดงแทนเมื่อรูปภาพโหลดไม่สำเร็จ
