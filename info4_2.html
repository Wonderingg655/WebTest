4.2 การจัดรูปแบบรูปภาพ
แอตทริบิวต์ style
  สามารถใข้แอตทริบิวต์ style ร่วมกับแท็กในภาษา HTML เช่น <p>, <h1>, <img>, <body> เพื่อกำหนดรูปแบบการแสดงผลขององค์ประกอบบนหน้าเว็บโดยตรง ด้วยการเขียนคำสั่ง CSS ภายในแท็ก HTML
การจัดรูปแบบข้อความมีตัวอย่าง ดังนี้
การปรับความสูงและความกว้างของรูปภาพ
  สามารถใช้พร็อบเพอร์ตี้ height เพื่อปรับความสูง และ width เพื่อปรับความกว้างของรูปภาพ ร่วมกับแอตทริบิวต์ style
คำสั่ง
ผลลัพธ์

คลิก เพื่อทดลองใช้งานด้วยบล็อกคำสั่ง
การปรับระยะห่างของรูปภาพ
  สามารถใช้พร็อบเพอร์ตี้ margin ร่วมกับแอตทริบิวต์ style เพื่อปรับระยะห่างของรูปภาพระหว่างองค์ประกอบอื่น ๆ ซึ่งในภาษา css จะมีพร็อบเพอร์ตี้โดยเฉพาะ เพื่อปรับระยะห่างทั้ง 4 ด้าน ได้แก่
  margin ปรับระยะห่างทั้ง 4 ด้าน
  margin-top ปรับระยะห่างด้านบน
  margin-right ปรับระยะห่างด้านขวา
  margin-bottom ปรับระยะห่างด้านล่าง
  margin-left ปรับระยะห่างด้านซ้าย

ผลลัพธ์

การใช้งานคำสั่งร่วมกัน
  สามารถใช้พร็อบเพอร์ตี้หลาย ๆ ตัวร่วมกันได้ตามความต้องการของผู้ใช้งาน โดยการเพิ่มพร็อบเพอร์ตี้ที่ต้องการภายในแอตทริบิวต์ style
